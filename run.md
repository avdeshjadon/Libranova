# How to Run the Project (Libranova)

This project consists of a Spring Boot backend and a React (Vite) frontend. You need to run both to get the full application working.

## 1. Backend (Spring Boot)

Open a terminal, navigate to the root directory of the project, and run the following Maven command:

```bash
mvn spring-boot:run
```

This will start the backend server, typically on `http://localhost:8080`.

## 2. Frontend (React / Vite)

Open a **new** terminal, navigate to the `frontend` directory, and start the React development server:

```bash
cd frontend
npm run dev
```

This will start the frontend development server, typically on `http://localhost:5173`.
